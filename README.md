# AI Resume & Job Match Analyzer

This is an educational IT project developed as part of the course "Fundamentals of Innovative IT Project Management".

The project aims to analyze a candidate's resume and compare it with job vacancy requirements using AI-based approaches.

## Key Features
- Resume upload and analysis
- Job vacancy comparison
- Match percentage calculation
- Skill gap identification
- Resume improvement recommendations

## Tools and Technologies
- Git & GitHub
- Jira (task management)
- Confluence (project documentation)
